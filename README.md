# Batteria Italia Frontend Angular

To build:
```
npm install
npm start
```

To build android app w/capacitor:
```
ng build
npx cap init
npx cap add android
npx cap copy android
npx cap sync android
npx cap open android
```

After every edit to refresh android App
```
npx cap sync android
```
